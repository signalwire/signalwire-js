### Major Fix: SDK Integration

**Fixed critical authentication bug in WelcomeModal.js:**

The previous implementation was using incorrect SDK API:
- `subscriberInfo$` - doesn't exist on SignalWire
- `inboundCall$` - doesn't exist on SignalWire
- `client.connect()` was being called manually but SDK already calls it automatically

**Corrected SDK usage:**
1. SignalWire constructor takes `{ token: string }` as credentials
2. SDK automatically calls `connect()` during initialization
3. Used Promise with `isConnected$` observable to wait for connection
4. Subscriber info accessed via `client.subscriber` object after connection
5. Incoming calls accessed via `client.session.incomingCalls$`

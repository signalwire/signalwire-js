Theming

Components consume the SignalWire DTCG token names directly (--bg-page, --fg-default, --interactive-button-primary-bg, --type-family-body, …). Override any subset on an ancestor element and everything downstream re-skins — including content inside shadow roots, since CSS custom properties inherit through them. How to author a theme

/* 1. Load the SignalWire token sheet (or write one with the same names). */
@import '@signalwire/web-components/theme.css';

/* 2. Override any subset of tokens on a wrapper element. */
[data-theme='fuchsia'] {
  --bg-page: #1a0a14;
  --fg-default: #fdf2f8;
  --interactive-button-primary-bg: var(--sw-fuchsia);
  --interactive-button-primary-hover: #d92866;
}

/* 3. Toggle on the document — values cascade into every shadow root. */
document.documentElement.dataset.theme = 'fuchsia';

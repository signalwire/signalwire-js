SignalWire Web Components

The stories are organized into four sections: low-level UI primitives, Layout containers, the turnkey Call Widget, and Devices helpers.

Try the call widget

Pick a credential in the top bar, then click the button to open it in modal mode. 

<sw-call-widget modal transcription
  token="YOUR_TOKEN"
  destination="/public/test-room">
  <sw-ui-background slot="background" default></sw-ui-background>
  <button>Open &amp; Call</button>
</sw-call-widget>

Embed — sw-click-to-call

The simplest SignalWire embed. One script tag + one custom element — no JavaScript required to get a working call button.

Zero-config usage

Drop this anywhere on your page. Clicking the button opens a modal call widget, dials the destination, and shows video + controls — all self-contained.

<!-- 1. Load the embed bundle (once, in <head>) --> <script src="/dist/embed/signalwire-web-components-embed.iife.js"></script> <!-- 2. Place the button anywhere in <body> --> <sw-click-to-call token="c2c_your_token_here" destination="/public/test-room" label="Call Us" ></sw-click-to-call>

CSS custom properties

The button colour, size, and shape are fully themeable via CSS custom properties. Set them on the element or any ancestor selector.

<sw-click-to-call token="c2c_…" destination="/public/test-room" label="Sales" style="--sw-c2c-bg:#7c3aed; --sw-c2c-bg-hover:#6d28d9; --sw-c2c-radius:100px;" ></sw-click-to-call>

Listening to events

sw-dial fires when a call starts. sw-hangup fires when it ends. Both bubble and cross shadow DOM — listen on the element or any ancestor.

<sw-click-to-call id="c2c" token="c2c_…" destination="/public/test-room"></sw-click-to-call> <script> document.getElementById('c2c').addEventListener('sw-dial', (e) => { console.log('call started', e.detail); }); document.getElementById('c2c').addEventListener('sw-call-hangup', () => { console.log('call ended'); }); </script>

Programmatic open / close

window.SignalWireUI exposes SignalWire, StaticCredentialProvider, and EmbedTokenCredentialProvider for scripts that need to do more than click a button. The widget's .dial() and .hangup() methods are also public.

// Classes available after the IIFE loads const { SignalWire, EmbedTokenCredentialProvider } = window.SignalWireUI; // Or call dial/hangup on the element directly document.querySelector('sw-click-to-call') .querySelector('sw-call-widget') .dial();

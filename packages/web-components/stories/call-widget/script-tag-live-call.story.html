Embed — sw-call-widget (real call)

sw-call-widget handles everything: client init, dialling, media, controls, and the optional AI transcript panel. Set token + destination and call .dial().

Inline mode

The widget fills its container. Wire up your own dial/hangup buttons — or use sw-click-to-call for a zero-JS drop-in.

<div style="width:760px; height:460px;"> <sw-call-widget id="widget" token="c2c_892b29eca7b6d96091faf713f07cdf46" destination="/public/test-room" transcription > <sw-ui-background slot="background" default></sw-ui-background> </sw-call-widget> </div> <script> document.getElementById('widget').dial(); // start call document.getElementById('widget').hangup(); // end call </script>

Modal mode — via sw-click-to-call

Set modal on sw-call-widget — or use the sw-click-to-call wrapper, which adds the modal + trigger button for you. Clicking the button dials and opens the overlay.

<sw-click-to-call token="c2c_892b29eca7b6d96091faf713f07cdf46" destination="/public/test-room" label="Open Call" ></sw-click-to-call> <!-- equivalent long form: --> <sw-call-widget modal token="c2c_…" destination="/public/test-room"> <button>Open & Call</button> <!-- any element in default slot triggers dial --> </sw-call-widget>

Receiving incoming calls

Add allow-incoming-calls to make the widget listen for inbound calls on the same token. A built-in prompt will appear to accept or reject.

<sw-call-widget token="c2c_892b29eca7b6d96091faf713f07cdf46" destination="/public/test-room" allow-incoming-calls transcription ></sw-call-widget>

Programmatic API

window.SignalWireUI exposes SDK classes so you can build a fully custom call flow without any UI at all — or mix-and-match with the widgets.

// Classes available after the embed bundle loads: const { SignalWire, EmbedTokenCredentialProvider, StaticCredentialProvider } = SignalWireUI; // Embed token (c2c_ / c2t_ prefix) — uses embeds.signalwire.com automatically: const client = new SignalWire( new EmbedTokenCredentialProvider('embeds.signalwire.com', 'c2c_892b...') ); // SAT token — point at your own project: const client2 = new SignalWire( new StaticCredentialProvider({ token: 'eyJ…your_SAT…' }) ); // Dial and wire context into a sw-call-provider: const call = await client.dial('/public/test-room', { audio: true, video: true }); document.getElementById('provider').call = call;

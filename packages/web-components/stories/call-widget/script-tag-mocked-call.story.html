Embed — sw-call-widget (mock / in-call UI)

Shows the full in-call layout wired to a fake video stream — no real SignalWire connection needed. Use this to develop and iterate on your embed UI.

How it works

The embed IIFE registers sw-call-provider, which is a transparent context wrapper. Set provider.call = call and every child component (sw-call-media, sw-call-controls, sw-local-camera, …) receives live state automatically — no manual property wiring.

<!-- 1. Embed bundle registers all custom elements --> <script src="/dist/embed/signalwire-web-components-embed.iife.js"></script> <!-- 2. Transparent context wrapper — children auto-receive call state --> <sw-call-provider id="provider"> <sw-ui-call-layout shadow> <sw-call-media slot="video"></sw-call-media> <sw-local-camera slot="floating-video" mirror></sw-local-camera> <sw-call-controls slot="controls" show-transcript></sw-call-controls> <sw-ui-transcript-view slot="transcript"></sw-ui-transcript-view> </sw-ui-call-layout> </sw-call-provider> <script> // One line to wire a call (real or mock) into the whole UI: document.getElementById('provider').call = myCall; </script>

Device selector — standalone

sw-device-selector works independently from any call. Pass a DeviceController-compatible object with observable lists and selection callbacks. After the mock call starts, the provider's device controller is also wired automatically.

<!-- The gear icon opens a popover with mic / camera / speaker sections --> <sw-device-selector id="devsel"></sw-device-selector> <script type="module"> import { BehaviorSubject } from 'rxjs'; const dc = { audioInputDevices$: new BehaviorSubject([]), videoInputDevices$: new BehaviorSubject([]), audioOutputDevices$: new BehaviorSubject([]), selectedAudioInputDevice$: new BehaviorSubject(null), selectedVideoInputDevice$: new BehaviorSubject(null), selectedAudioOutputDevice$: new BehaviorSubject(null), selectAudioInputDevice: (d) => { /* update selection */ }, selectVideoInputDevice: (d) => { /* update selection */ }, selectAudioOutputDevice: (d) => { /* update selection */ }, }; await navigator.mediaDevices.getUserMedia({ audio: true }); // grant permission first const devices = await navigator.mediaDevices.enumerateDevices(); dc.audioInputDevices$.next( devices.filter(d => d.kind === 'audioinput')); dc.videoInputDevices$.next( devices.filter(d => d.kind === 'videoinput')); dc.audioOutputDevices$.next(devices.filter(d => d.kind === 'audiooutput')); document.getElementById('devsel').deviceController = dc; </script>

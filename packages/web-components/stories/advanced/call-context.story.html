Context Provider

Wrap your UI in <sw-call-provider> and assign provider.call = call after dialing. Every child component picks up the call automatically.

Key lines 
// 1. Create the client and connect
client = new SignalWire(new StaticCredentialProvider({ token }));
await waitForConnection(client);

// 2. Dial
call = await client.dial(destination, { audio: true, video: true, receiveAudio: true, receiveVideo: true });

// 3. Push the call into Lit context — this is the only wiring needed.
//    sw-call-media, sw-call-status, etc. wake up automatically.
provider.call = call;

// 4. Tear down
call.hangup();
client.destroy();
provider.call = null;
 
Enter a SAT token and press Start Call.

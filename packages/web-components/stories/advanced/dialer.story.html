Simple Dialer

<sw-call-dialpad allow-text> accepts free-text destinations (e.g. SIP URIs, room names). On sw-dial this story creates a call and pushes it into <sw-call-provider> — every child component wakes up automatically. Once connected, keypad digits are sent as DTMF by sw-call-dialpad.

Key lines 
// Listen for dial events from the text-mode dialpad
dialpad.addEventListener('sw-dial', async (e) => {
  const destination = e.detail.digits;

  client = new SignalWire(new StaticCredentialProvider({ token }));
  await waitForConnection(client);

  call = await client.dial(destination, {
    audio: true, video: true, receiveAudio: true, receiveVideo: true,
  });

  // Push into context — sw-call-media, sw-call-status, etc. wake up.
  provider.call = call;
});
 
Enter a SAT token above, type a destination, press Call.

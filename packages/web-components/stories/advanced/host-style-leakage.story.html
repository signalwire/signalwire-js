Slotted content (intentionally inherits)

Slotted nodes inherit from the flat-tree parent (the host page), not from :host of the component they're slotted into. This is correct: user-provided slot markup keeps the page's typography. The <button> below visibly takes on the hostile styles when toggled, while the widget's own chrome does not.

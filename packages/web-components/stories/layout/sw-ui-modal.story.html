sw-ui-modal

Animations are exposed as CSS custom properties: --sw-modal-animation, --sw-modal-close-animation, --sw-modal-backdrop-animation, --sw-modal-backdrop-close-animation. Override per-instance to swap motion.

Variants

Default — bounce

Built-in spring on open + close.

Slide up

Sheet-style entrance from below.

Fade

Pure opacity, no movement.

Zoom

Scale in from 50% with spring easing.

3D flip

Rotates in on the X axis.

Slow drift

Cinematic blur + lift, custom backdrop fade.

Snippet

<style>
  sw-ui-modal#slide {
    --sw-modal-duration: 0.3s;
    --sw-modal-animation: slide-up var(--sw-modal-duration) cubic-bezier(0.32, 0.72, 0, 1);
    --sw-modal-close-animation: slide-down var(--sw-modal-duration) cubic-bezier(0.32, 0.72, 0, 1);
  }
  @keyframes slide-up   { from {transform: translateY(100vh); opacity: 0} to {transform: translateY(0); opacity: 1} }
  @keyframes slide-down { from {transform: translateY(0); opacity: 1}  to {transform: translateY(100vh); opacity: 0} }
</style>

<sw-ui-modal id="slide">…</sw-ui-modal>

Default

The built-in bounce-in / bounce-out animation.

Slide up

Slides up from the bottom of the viewport, like a mobile bottom sheet.

Fade

Just opacity transition. No movement.

Zoom

Scales up from 50% with a spring curve.

3D flip

Flips in on the X axis with a slight overshoot.

Slow drift

Custom backdrop fade pairs with a longer, blurred entry animation.
